Guide Complet CSS - Transitions & Animations

Tout ce que vous devez savoir sur les transitions et animations CSS

Introduction aux Transitions CSS

Les transitions CSS permettent de créer des animations fluides entre les différents états d'un élément. Il existe deux méthodes principales pour les implémenter.

Rappel : Les transitions nécessitent toujours un déclencheur comme :hover, :focus, ou des changements de classe via JavaScript.

Méthode 1: Syntaxe Raccourcie (Shorthand)

La méthode raccourcie permet de définir toutes les propriétés de transition en une seule ligne.

Syntaxe Raccourcie

.element { transition: all 0.5s ease; } .element:hover { background: #e74c3c; transform: scale(1.2); border-radius: 50%; }
Hover Me

Structure de la Syntaxe

transition: [property] [duration] [timing-function] [delay];
Exemples :
transition: all 0.3s;
transition: opacity 1s ease-in;
transition: transform 0.5s ease-out 0.2s;
transition: width 2s, height 1s;

Méthode 2: Propriétés Séparées

Vous pouvez également définir chaque propriété de transition individuellement.

Propriétés Séparées

.element { transition-property: color, top, margin; transition-duration: 1s, 5s, 2s; } .element:hover { color: yellow; margin-top: 30px; background: #9b59b6; }
Hover Me

Les Quatre Propriétés

transition-property

Définit quelles propriétés CSS seront animées

transition-property: color, transform, opacity;
transition-duration

Définit la durée de l'animation pour chaque propriété

transition-duration: 1s, 5s, 2s;
Important : L'ordre des durées dans transition-duration doit correspondre à l'ordre des propriétés dans transition-property.

Fonctions de Temporisation (Timing Functions)

Les fonctions de temporisation contrôlent la vitesse de l'animation durant sa durée.

ease (par défaut)

Démarrage lent, rapide au milieu, fin lent

Hover

linear

Vitesse constante du début à la fin

Hover

ease-in

Démarrage lent, fin rapide

Hover

ease-out

Démarrage rapide, fin lente

Hover

ease-in-out

Démarrage et fin lents, rapide au milieu

Hover
Utilisation avec les propriétés séparées :
transition-timing-function: ease-in;
transition-timing-function: ease, linear, ease-in-out;

Synthèse : Les Quatre Propriétés Réduites à Une

Les quatre propriétés de transition peuvent être combinées en une seule déclaration dans cet ordre :

Ordre de la syntaxe raccourcie :
transition: [property] [duration] [timing-function] [delay];

Propriétés Séparées

.element { transition-property: transform, background-color; transition-duration: 1.5s, 2s; transition-timing-function: ease-in-out, linear; transition-delay: 0.5s, 0s; }

Syntaxe Raccourcie Équivalente

.element { transition: transform 1.5s ease-in-out 0.5s, background-color 2s linear; }
Hover Me
Règle importante : Dans la syntaxe raccourcie, l'ordre des valeurs doit toujours être :
property → duration → timing-function → delay

Introduction aux Animations CSS

Les animations CSS permettent de créer des effets visuels complexes en utilisant la règle @keyframes pour définir la séquence d'animation.

Structure de base :
@keyframes nomAnimation { ... }
element { animation: nomAnimation durée timing-function délai iteration-count direction; }

Animations de Base

Animations simples pour débuter avec les keyframes CSS.

Pulsation

Pulse
@keyframes pulsation { 0% { transform: scale(1); } 50% { transform: scale(1.1); } 100% { transform: scale(1); } } .element { animation: pulsation 2s infinite; }

Déplacement

Déplace
@keyframes deplacement { 0% { transform: translateX(0); } 25% { transform: translateX(50px); } 50% { transform: translateX(50px) translateY(50px); } 75% { transform: translateX(0) translateY(50px); } 100% { transform: translateX(0) translateY(0); } } .element { animation: deplacement 4s infinite; }

Rotation

Tourne
@keyframes rotation { 0% { transform: rotate(0deg); } 100% { transform: rotate(360deg); } } .element { animation: rotation 3s linear infinite; }

Changement de Couleur

Couleurs
@keyframes changementCouleur { 0% { background-color: #e74c3c; } 25% { background-color: #3498db; } 50% { background-color: #2ecc71; } 75% { background-color: #f39c12; } 100% { background-color: #9b59b6; } } .element { animation: changementCouleur 5s infinite; }

Voici un exemple concret montrant comment utiliser les propriétés d'animation avec des fichiers HTML et CSS séparés.

Animation avec Délai, Répétition et Nom

Fichier HTML (index.html)

<!DOCTYPE html>
<html lang="fr">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Exemple Animation</title>
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <div class="container">
        <div class="box demo-delay">Délai 1s</div>
        <div class="box demo-iteration">3 Répétitions</div>
        <div class="box demo-name">Nom Animation</div>
        <div class="box demo-combined">Combiné</div>
    </div>
</body>
</html>

Fichier CSS (styles.css)

/* Définition de l'animation */
@keyframes bounceAndColor {
    0% {
        transform: translateY(0) scale(1);
        background-color: #3498db;
        border-radius: 8px;
    }
    25% {
        transform: translateY(-20px) scale(1.1);
        background-color: #e74c3c;
        border-radius: 20px;
    }
    50% {
        transform: translateY(0) scale(1.2);
        background-color: #2ecc71;
        border-radius: 35px;
    }
    75% {
        transform: translateY(-10px) scale(1.1);
        background-color: #f39c12;
        border-radius: 20px;
    }
    100% {
        transform: translateY(0) scale(1);
        background-color: #3498db;
        border-radius: 8px;
    }
}

/* Styles de base */
.container {
    display: flex;
    justify-content: space-around;
    align-items: center;
    flex-wrap: wrap;
    gap: 20px;
    padding: 20px;
}

.box {
    width: 120px;
    height: 120px;
    background: #3498db;
    border-radius: 8px;
    display: flex;
    align-items: center;
    justify-content: center;
    color: white;
    font-weight: bold;
    text-align: center;
    box-shadow: 0 4px 8px rgba(0,0,0,0.1);
}

/* Animation avec délai */
.demo-delay {
    animation-name: bounceAndColor;
    animation-duration: 2s;
    animation-delay: 1s;
    animation-iteration-count: infinite;
}

/* Animation avec nombre limité de répétitions */
.demo-iteration {
    animation-name: bounceAndColor;
    animation-duration: 2s;
    animation-iteration-count: 3;
}

/* Animation avec nom spécifique */
.demo-name {
    animation-name: bounceAndColor;
    animation-duration: 2s;
    animation-iteration-count: infinite;
}

/* Animation combinant plusieurs propriétés */
.demo-combined {
    animation-name: bounceAndColor;
    animation-duration: 2s;
    animation-delay: 0.5s;
    animation-iteration-count: 5;
}
Explication des propriétés :
  • animation-name : Définit le nom de l'animation à appliquer (doit correspondre au nom dans @keyframes)
  • animation-delay : Définit le délai avant le début de l'animation
  • animation-iteration-count : Définit le nombre de répétitions de l'animation (peut être un nombre ou "infinite")

Transitions vs Animations

Comprendre quand utiliser les transitions et quand utiliser les animations.

Transitions CSS

  • ✅ Simple à mettre en œuvre
  • ✅ Parfait pour les changements d'état simples
  • ✅ Animation automatique inverse
  • ❌ Pas de contrôle sur les états intermédiaires
  • ❌ Pas de boucle intégrée
  • ❌ Nécessite un déclencheur
Cas d'utilisation :
Boutons au survol
Menus déroulants
Changements de couleurs
Animations simples

Animations CSS

  • ✅ Contrôle total avec @keyframes
  • ✅ Boucles infinies possibles
  • ✅ Pas besoin de déclencheur
  • ✅ États intermédiaires personnalisables
  • ❌ Plus complexe à mettre en place
  • ❌ Pas d'animation inverse automatique
Cas d'utilisation :
Spinners de chargement
Animations complexes
Effets continus
Séquences d'animation
Recommandation : Utilisez les transitions pour les interactions utilisateur simples et les animations pour les effets visuels complexes et automatiques.

Tableau Comparatif

Transitions

/* Syntaxe */ transition: property durée timing délai; /* Exemple */ transition: all 0.3s ease; /* Déclenchement */ .element:hover { /* styles changés */ }

Animations

/* Syntaxe */ @keyframes nom { 0% { /* état initial */ } 100% { /* état final */ } } .element { animation: nom durée timing délai count direction; } /* Déclenchement */ /* Automatique ou via classe */

Exemples Avancés

Combinaisons créatives de transitions et d'animations.

Spinner de Chargement

@keyframes chargement { 0% { transform: rotate(0deg); } 100% { transform: rotate(360deg); } } .element { animation: chargement 1.5s linear infinite; border: 5px solid #f3f3f3; border-top: 5px solid #3498db; border-radius: 50%; }

Effet de Vibration

Vibre
@keyframes vibration { 0%, 100% { transform: translateX(0); } 10%, 30%, 50%, 70%, 90% { transform: translateX(-5px); } 20%, 40%, 60%, 80% { transform: translateX(5px); } } .element { animation: vibration 0.5s infinite; }

Animation d'Apparition

Apparaît
@keyframes croissance { 0% { transform: scale(0.5); opacity: 0; } 70% { transform: scale(1.1); opacity: 1; } 100% { transform: scale(1); opacity: 1; } } .element { animation: croissance 1s forwards; }

Dégradé Mouvant

Dégradé
@keyframes degradeMouvant { 0% { background-position: 0% 50%; } 50% { background-position: 100% 50%; } 100% { background-position: 0% 50%; } } .element { background: linear-gradient(270deg, #ff6b6b, #ffa726, #2ecc71, #3498db); background-size: 800% 800%; animation: degradeMouvant 8s ease infinite; }

Bonnes Pratiques

Conseils pour des animations performantes :
  • Utilisez transform et opacity pour de meilleures performances
  • Évitez d'animer les propriétés width, height, margin, padding
  • Utilisez will-change pour les éléments qui seront animés
  • Testez vos animations sur différents appareils et navigateurs
  • Pensez à l'accessibilité (réduire les animations pour les utilisateurs sensibles)
Structure complète d'une animation :
animation: [name] [duration] [timing-function] [delay] [iteration-count] [direction] [fill-mode] [play-state];
